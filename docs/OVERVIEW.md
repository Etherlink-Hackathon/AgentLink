# AgentLink

**AgentLink** is an automated DeFi yield protocol on the Etherlink Mainnet. It combines **Gemini-driven AI strategists** with an **ERC-4626 Arbitrage Vault** to generate sustainable yield through trustless, on-chain execution.

## Core Pillars

- **The Vault**: An ERC-4626 compliant contract that manages user deposits and enforces strict on-chain guardrails.
- **The Strategist**: A sophisticated AI agent that monitors DEX liquidity pools (Curve, Oku/UniV3) to capture arbitrage spreads.
- **Capital Safety**: All trades are atomic and include mandatory profit enforcement logic.

## Project Structure

| Component | Role |
| :--- | :--- |
| **Frontend** | React-based yield dashboard for LPs. |
| **Contracts** | Solidity-based ERC-4626 vaults and router integrations. |
